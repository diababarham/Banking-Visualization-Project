# Banking-Visualization-Project
This is a Power Bi project that aims to show visualization for a bank's transactions. It is divided into 5 pages, with the first page explaining each page with a short description. The goal of the project is to provide tangible findings of the customer base. 

# Intro-Page
This is the introduction page of the Bank Performance Analysis dashboard. It has a clear overview of the project’s purpose and serves as a navigation hub by having buttons on the left that are bookmarks to the rest of the pages. Do make sure to use CTRL + Left click to navigate. The page outlines the four main analytical sections of the report.
